How to Use

  1. Upload Image: Select an image file from your device or load a sample
  2. Choose Aspect Ratio: Select a preset ratio or use "Free" for custom cropping
  3. Set Dimensions: Enter specific width and height values if needed
  4. Adjust Quality: Use the slider to control output quality (higher = larger file)
  5. Crop & Download: Click "Crop Image" to process and download the result

Supported Formats

Input: JPEG, PNG, WebP, GIF, BMP | Output: JPEG, PNG
